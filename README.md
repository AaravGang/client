## Setup

### Preprequisites

- python3.8 or python3.9
- brain

> Download this folder and unzip
> Next open terminal and follow these commands

**For Mac/Linux users**

```
cd <path to download>
python3 -m pip install -r requirements.txt
python3 client.py
```

**For Windows**

```
cd <path to download>
python -m pip install -r requirements.txt
python client.py
```
